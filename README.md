# Credit-Card-Customer-Transaction-Analysis
Project Overview

This project demonstrates an end-to-end Business Intelligence workflow using Python, PostgreSQL, SQL, Power BI, and Excel.

The objective of this project is to analyze customer behavior, credit card transactions, revenue trends, customer acquisition cost, and business performance by transforming raw data into meaningful business insights.

Objectives:

* Clean and preprocess raw customer & transaction data
* Store data inside PostgreSQL
* Perform business analysis using SQL
* Build interactive dashboards in Power BI

Key Business Insights:

* Blue Card generated the highest revenue.
* Swipe transactions contributed the highest transaction amount.
* Graduate customers generated the highest revenue.
* Q3 recorded the highest transaction count.
* Businessman and Self-employed customers generated higher revenue.
* Revenue varied significantly across customer categories.
